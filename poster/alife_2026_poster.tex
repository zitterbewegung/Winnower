\LeadBox{\textbf{Main result.} A single Bernoulli-NML selector fits a global relative-periodic scaffold to finite spacetimes in 1D, 2D, and 3D cellular automata. In the strongest 2D cases, the remaining defect mask is sparse, but it does not collapse to featureless noise.}

\SectionHeader{Problem}
Cellular automata often contain broad domains together with particles, interfaces, and other persistent defects. For a finite spacetime, the practical question is which global period and shift give the best explanation of the observed pattern.

\LeadBox{\textbf{Guiding idea.} First fit the global scaffold, and then inspect the residual mask that remains.}

\SectionHeader{Selector}
\begin{itemize}
  \item A candidate $(p,\mathbf{s})$ defines a relative-periodic model class and its orbit partition.
  \item Majority vote on each orbit yields the fitted background $B^*(p,\mathbf{s})$.
  \item The residual mask is $M = U \oplus B^*(p,\mathbf{s})$, and Bernoulli NML ranks the candidates.
\end{itemize}

\[
\mathcal{B}(p,\mathbf{s}) =
\left\{
B :
B[t+p,\mathbf{x}+\mathbf{s} \bmod \mathbf{D}] = B[t,\mathbf{x}]
\right\}
\]

\[
\mathrm{NML}(p,\mathbf{s}) =
\sum_j n_j H_b(\hat{\theta}_j)
\;+\;
\frac{1}{2}\sum_j \log_2 n_j
\]

Here $n_j$ is the size of orbit class $j$ and $\hat{\theta}_j$ is its fitted Bernoulli parameter. Pure orbit classes lower the fit term, but extra template bits raise the penalty.

Raw Bernoulli likelihood rewards large templates because they can memorize the data. The NML penalty removes that failure mode.

\SectionHeader{Focal 2D Cases}
\begin{itemize}
  \item In \textbf{Diamoeba}, large breathing domains lead the selector to choose $(p,\mathbf{s})=(1,(0,0))$, with defect rate $0.077$.
  \item In \textbf{Maze with Mice}, a coherent maze background yields winner $(2,(0,0))$, with defect rate $0.008$.
  \item In \textbf{S37/B11}, a low-defect background coexists with persistent residual structure; the presentation run gives winner $(2,(0,0))$ and defect rate $0.015$.
\end{itemize}

\SectionHeader{Stability Across Horizon}
As the observation horizon grows, the selected period usually stabilizes quickly.

\PosterCard{
  \vspace{0.10em}
  \includegraphics[width=\linewidth]{stabilization_summary.pdf}
  \vspace{0.04em}
}

\begin{itemize}
  \item \textbf{ECA-54} locks immediately to period $4$.
  \item \textbf{ECA-110} passes through a short transient and then locks to period $4$ with shift $-2$.
  \item \textbf{S24/B11} changes later, because longer horizons reveal additional periodic structure.
  \item \textbf{Diamoeba3d} corrects a small-sample transient and then locks to period $1$.
\end{itemize}

\SectionHeader{Takeaways}
\LeadBox{
\begin{itemize}
  \item One Bernoulli-NML selector provides usable periodic scaffolds in 1D, 2D, and 3D cellular automata.
  \item The complexity penalty changes the selected period qualitatively; it is not a cosmetic adjustment.
  \item In 2D, residual masks expose structured defects and therefore make broader surveys feasible.
  \item The selector is a front end for richer local analysis, not a replacement for computational mechanics.
\end{itemize}
}
